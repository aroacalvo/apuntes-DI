# Stream prozesamendua

a data processing method where data is processed continuously in real time as it is generated, instead of waiting to group them into batches.

## Ezaugarriak/abantailak

1. Denbora Erreala
2. Jarraitua (datuak iritsi ahala prozesatu)
3. Eskalagarria
4. Malgua (datu egitaratuak eta egitaratu gabeak)

## Desabantailak

1. Konplexutasuna (sistemaren diseinua, erroreak kudeatu, datuen kalitatea bermatzea)
2. Egoeraren kudeaketa (aplikazioen egoera mantentzea zaila)
3. Baliabide beharrak

## Erramintak

- Apache Kafka Streams
- Apache Spark
- Apache Flink
- Apache Storm
