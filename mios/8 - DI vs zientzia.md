# Datuen Ingeniaritza vs Datuen Zientzia

- Data Engineering → building and maintaining the infrastructure and pipelines to collect, store, and move data.
- Data Science → analyzing and modeling data to extract insights, make predictions, and inform decisions.

| Feature              | Data Engineering                               | Data Science                                |
| -------------------- | ---------------------------------------------- | ------------------------------------------- |
| **Focus**            | Infrastructure, pipelines, storage             | Analysis, modeling, insight generation      |
| **Goal**             | Ensure data is clean, reliable, and accessible | Turn data into actionable knowledge         |
| **Tools**            | SQL, ETL tools, Hadoop, Spark, Airflow, Kafka  | Python, R, pandas, scikit-learn, TensorFlow |
| **Skills**           | Programming, databases, distributed systems    | Statistics, ML, data visualization          |
| **Output**           | Data pipelines, warehouses, ETL processes      | Reports, dashboards, ML models              |
| **Role in workflow** | Prepares data for use                          | Uses data to answer business questions      |
