# Motibazioa

## Datu ingeniaritza

- Datuen Ingeniaritza, datu gordinak kalitate altuko informazioan ekoizten dituzten sistema eta prozesuen garapen, inplementazio eta mantenimendua da, datuen analisia eta machine learning bezalako erabilerak jasanez
- Datuen Ingeniaritza, segurtasun, datu kudeaketa, DataOps, datuen arkitektura, orkestrazio eta software ingeneritza arloen elkargunea da
- Datu ingeniariak, datuen bizi-zikloa kudeatzen du, datuak jatorritik jasotzetik erabilera ezberdinei eskaini arte, analisia edo machine learning esaterako
