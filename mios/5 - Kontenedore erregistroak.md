# Kontenedore erregistroak

- Kontenedore irudiak gordetzen dituen gordailu edo biltegia
- Kontenedoreak gorde, antolatu, partekatu eta banatzeko aukera ematen dute
- Like GitHub for containers

## Ezaugarri nagusiak

1. Irudien biltegiratze eta kudeaketa
   - Kontenedore irudien bertsioak kudeatu, igo eta gorde daitezke
2. Sarrera kontrola
   - Nork zer egin dezakeen kudeatu daiteke
3. Interfaze uniformea
   - Normalean linea komando bidez edo API REST baten bitartez konektatu gaitezke
4. Orkestazio eta garapen erramintekin integrazioa
   - Kontenedoreak martxan jarri eta aplikazioak kudeatzeko hainbat erramintekin integratzen dira
5. Segurtasuna
   - Hainbat segurtasun ezaugarri izaten dituzte, hala nola kontenedore irudien segurtasun akatsak ikuskatzeko aukera

## Motak

1. Erregistro publikoak
2. Erregistro pribatuak
3. Zerbitzu hornitzaileek eskainitako erregsitroak
   - Hodei zerbitzu hornitzaile askok beraien plataformetan txertatutako kontenedore erregistroakeskaintzen dituzte
