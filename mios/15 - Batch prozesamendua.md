# Batch prozesamendua

A data processing method where large volumes of data are collected and processed together at scheduled times, rather than in real time.

Noiz:

- Data bolumen handiak
- Eragiketa errepikakorrak
- Modo automatizatuan egiteko

Erramintak:

- Apache Hadoop
- Apache Spark
- Apache Flink

## Abantailak

1. Automatizazioa (ordutegi batean exekutatu irizpide jakin batzuk bete ondoren)
2. Eskalabilitatea
3. Eraginkortasuna (taldekatzea -> optimizatu)
4. Independentzia (gainbegiratu gabe)

## Desabantailak

1. Erantzun denbora
2. Konplexutasuna akatsen kudeaketan
